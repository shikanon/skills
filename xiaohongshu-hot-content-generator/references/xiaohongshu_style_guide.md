# 小红书热门内容风格指南

## 文案风格要求
1. 开头要有吸引力：使用emoji、提问式、利益点前置，比如「谁懂啊！」「我宣布！」「亲测有效！」
2. 语气亲切，像朋友分享，避免官方话术，多用口语化表达
3. 结构清晰，多用小标题、emoji分隔，重点内容加粗
4. 结尾要有互动引导，比如「你们觉得怎么样？」「快去试试！」「评论区交流呀～」
5. 字数控制在300-800字，不要太长

## 图片风格要求
1. 整体明亮、清新、饱和度适中，符合小红书审美
2. 4K分辨率，比例3:4（小红书最优比例）
3. 图片有清晰的主体，信息明确，避免杂乱
4. 优先选择ins风、简约风、可爱风，根据内容调整
5. 可以添加适当的文字 overlay 突出重点，但不要遮挡主体

## 标签要求
1. 标签数量控制在5-10个
2. 包含1-2个大流量标签（比如#OpenClaw #AI工具 #创业）
3. 包含3-5个垂直标签（比如#AI变现 #副业 #工具推荐）
4. 包含2-3个长尾标签（比如#普通人创业 #2025副业方向 #AI工具赚钱）
5. 标签放在文案最后，用空格分隔

## 热门选题方向
1. 工具测评类：「亲测10款AI工具，只有这3款能赚钱」
2. 经验分享类：「我用OpenClaw做副业，第一个月赚了8000」
3. 教程类：「3步教你用OpenClaw搭建自己的AI助手，小白也能会」
4. 避坑类：「做AI变现踩过的5个坑，第一个就亏了2000」
5. 对比类：「AI工具大比拼，OpenClaw vs 其他同类工具优势在哪」
